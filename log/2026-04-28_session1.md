## 目的・要件

- お気に入りのWebサイト・ツール・アプリをジャンルごとに整理できるページを作りたい
- PC・スマートフォンなど複数デバイスから同じデータを使いたい
- 自分だけが使えるプライベートなものにしたい
- Claude Code 初心者のため、手順を丁寧に解説しながら進める

---

## 採用した方法

**方法B：静的ウェブページ（GitHub Pages）**

| 項目 | 内容 |
|------|------|
| ホスティング | GitHub Pages（無料） |
| リポジトリ | Public（無料枠の制約） |
| プライバシー | アプリ内パスワード保護（SHA-256ハッシュ） |
| データ同期 | GitHub API（Personal Access Token）で `data.json` を読み書き |

---

## 作成したファイル

### `index.html`
ページの骨格。ロック画面・ジャンルタブ・カードグリッド・各種モーダル（お気に入り追加/編集、ジャンル追加/編集、設定）を含む。

### `style.css`
モダンなカードデザイン。CSS変数でテーマ管理。PC・スマホ両対応のレスポンシブレイアウト。

### `app.js`
全機能のロジック。主な機能：
- パスワードロック（Web Crypto API で SHA-256 ハッシュ化、localStorage に保存）
- お気に入りのCRUD（追加・編集・削除）
- ジャンルのCRUD（追加・編集・削除）
- ドラッグ&ドロップ並び替え（カード・ジャンルタブ両対応）
- GitHub API 経由でのデータ同期（push / pull）
- データは localStorage に保存（オフライン対応）

### `data.json`
お気に入りデータの実体。GitHub Pages から配信され、GitHub API で更新される。

```json
{
  "genres": [...],
  "bookmarks": [...]
}
```

### `.gitignore`
`.claude/` ディレクトリ（Claude Code の内部設定）を除外。

## 実装した機能一覧

| 機能 | 説明 |
|------|------|
| パスワードロック | 初回起動時にパスワード設定。以降はハッシュ比較で認証 |
| ジャンルタブ | タブをクリックして切り替え。ダブルクリックで編集 |
| カード表示 | Google Favicon API でアイコン表示。タイトル・説明・ドメイン名 |
| お気に入り追加 | URL・タイトル・説明・ジャンルを入力して保存 |
| 編集・削除 | カードの ✎ ボタンから編集モーダルを開く |
| ドラッグ&ドロップ | カードもジャンルタブもドラッグで並び替え |
| GitHub同期（push） | 「⟳ 同期」ボタンで現在のデータを GitHub に保存 |
| GitHub同期（pull） | 設定画面の「GitHubから読み込む」で別デバイスのデータを取得 |
| パスワード変更 | 設定画面から変更可能 |
| ロック | ヘッダーの 🔒 ボタンでいつでもロック可能 |

## セットアップ手順（実施済み）

1. `git init` でローカルリポジトリ作成
2. ファイルを追加してコミット
3. GitHub に `my-favorites` リポジトリを作成（最初 Private → Public に変更）
4. `git push -u origin main` でアップロード
5. GitHub Settings → Pages → Branch: `main` で公開設定
6. サイトにアクセスしてパスワード設定
7. GitHub → Settings → Personal Access Tokens で fine-grained トークン作成
   - 対象リポジトリ：`setoband3/my-favorites`
   - 権限：Contents → Read and write
8. アプリの設定画面にトークンを入力・保存
9. 同期テストで `data.json` が更新されることを確認

---

## GitHub Pages の制約（判明した問題）

- 無料アカウントでは **Private リポジトリに GitHub Pages を設定できない**
- 対策：リポジトリを Public にして、アプリ側のパスワード保護でプライバシーを確保

---

## 別デバイス（スマートフォン）での使い方

1. ブラウザで `https://setoband3.github.io/my-favorites/` を開く
2. 初回：パスワードを設定
3. 設定 → GitHub同期設定 にユーザー名・リポジトリ名・トークンを入力
4. 「GitHubから読み込む」をタップ → PCのデータが反映される
5. ホーム画面に追加するとアプリ風に使える（Safari：共有→ホーム画面に追加）
